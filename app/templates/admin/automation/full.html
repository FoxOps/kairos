Comment ça marche ?

Cette page permet de générer simultanément les astreintes et les shifts pour une période donnée.

  1. Configurez la période : Sélectionnez la date de début et de fin.
  2. Configurez la rotation des astreintes : Sélectionnez les utilisateurs et leur ordre.
  3. Configurez les règles des shifts : Définissez le nombre de shifts par type et par jour.
  4. Prévisualisez : Utilisez le bouton "Prévisualiser" pour voir ce qui sera généré.
  5. Générez : Cliquez sur "Générer tout" pour créer les astreintes et les shifts en base de données.

Note : Les règles de remplacement automatique s'appliquent pour les deux types de génération.
