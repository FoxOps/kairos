Comment ça marche ?

  1. Sélectionnez l'ordre de rotation : Cochez les utilisateurs que vous souhaitez inclure dans la rotation et définissez leur ordre.
  2. Choisissez la période : Sélectionnez la date de début et de fin pour la génération.
  3. Prévisualisez : Utilisez le bouton "Prévisualiser" pour voir ce qui sera généré sans sauvegarder.
  4. Générez : Cliquez sur "Générer les astreintes" pour créer les astreintes en base de données.

Règles appliquées :

  • Chaque astreinte commence le vendredi à 21h et se termine le vendredi suivant à 07h.
  • La rotation suit l'ordre que vous avez défini.
  • Si un utilisateur n'est pas disponible (congé, astreinte existante), le système passe au suivant dans la rotation.
  • Un message d'avertissement est affiché si aucun utilisateur n'est disponible pour une période.
