Rafraîchir les Shifts

Cette page permet de recalculer tous les shifts pour une période donnée en tenant compte des astreintes actuelles (même modifiées manuellement).

Les nouvelles règles métiers seront appliquées :

  • Créneau 13h-21h pour la personne d'astreinte (si elle fait partie d'un groupe schedule)
  • Rotation : 13h-21h → 07h-15h la semaine suivante
  • 09h-17h par défaut pour les autres
  • Cas spécial avec 2 personnes : la non-astreinte passe en 07h-15h
  • Contrainte légale : 2 semaines minimum entre deux astreintes
