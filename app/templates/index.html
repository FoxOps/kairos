Conseils d'utilisation :

  • Mode édition activé : Le drag & drop est activé. Cliquez sur un événement (shift, astreinte, congé) pour le supprimer (avec confirmation).
  • Déplacer un événement : Cliquez et faites glisser un événement (shift, astreinte, congé) vers une nouvelle date/heure

  • Mode édition désactivé : Le drag & drop est désactivé. Cliquer sur un événement ne fait rien.
